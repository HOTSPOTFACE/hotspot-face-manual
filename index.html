Hotspot-Face
Touchscreen MMDVM monitor for DMR hotspots running PI-Star.

1. Introduction

The Hotspot-Face is an advanced touchscreen monitor specifically designed for DMR hotspot systems operating PI-Star.

It provides real-time visibility of transmissions, talkgroups, callsigns, and activity.

2. Features

  • 2.8" TFT touchscreen
  • Real-time DMR activity
  • Color-coded status indicators
  • Auto network mode switching
  • Auto-discovery for PI-Star

3. Initial Setup

  1. Power on the Hotspot-Face.
  2. Connect to WiFi: Hotspot-FACE Setup
  3. Go to: 192.168.4.1
  4. Enter WiFi + PI-Star info.
  5. Save and reboot.
Tip: Auto-discovery will find your PI-Star automatically.

4. Display Layout

Header Device name + settings access
Status Idle / TX / Error
Callsign Active caller

Status Indicators

  • Idle: No transmission
  • TX: Transmission active
  • No PI-Star: Cannot reach PI-Star
  • Connected successfully
  • Connection failed

5. Normal Operation

  • Auto network switching
  • Auto PI-Star connection
  • Live display updates

6. Settings Menu

  • Change device name
  • Change WiFi
  • Update PI-Star IPs
  • Toggle Auto-Discovery

7. Advanced Features

Auto-discovery, IP fallback, connection recovery.

8. Factory Reset

This erases all saved settings.
  1. Press & hold header for 10 seconds.
