Hotspot-Face User Manual
Advanced MMDVM Monitor for DMR hotspot systems operating PI-Star
Hotspot-Face
Touchscreen MMDVM monitor for DMR hotspots running PI-Star.
Created by: Robert Eddy (M3URE) ¬∑ Use case: Always-on visual monitor for PI-Star activity.

1. Introduction

The Hotspot-Face is an advanced touchscreen monitor specifically designed for DMR hotspot systems operating PI-Star. It provides real-time activity monitoring without needing a PC or mobile device.

Callsigns, talk groups, connection status, and TX duration appear instantly on a clean, dedicated display for your shack.

2. Features

Display

  • 2.8" TFT touchscreen (320√ó240 resolution)
  • Colour-coded status indicators
  • Real-time DMR activity display
  • Date and connection mode indicator

Network

  • 2.4GHz WiFi support
  • Automatic WiFi vs Hotspot mode detection
  • Auto-discovery of PI-Star devices
  • Web configuration interface

Configuration

  • Touchscreen menu system
  • On-screen keyboard
  • Captive setup portal
  • Factory reset functionality

3. Initial Setup

When powered on for the first time, the device enters WiFi Setup Mode.

  1. Power the device via USB-C.
  2. Connect to WiFi: Hotspot-FACE Setup
    Password: M3URE1512
  3. Open a browser and go to 192.168.4.1.
  4. Enter WiFi SSID + Password.
  5. Enter PI-Star WiFi IP and PI-Star Hotspot IP.
  6. Enable Auto-Discovery if unsure of IPs.
  7. Save & reboot.

5. Normal Operation

  • Automatically connects to WiFi
  • Detects WiFi or Hotspot mode
  • Fetches PI-Star data every second
  • Displays callsign, TG, TX duration

8. Factory Reset

Warning: Factory reset will erase all settings.
  1. Hold the header bar
  2. Wait for countdown
  3. Release to reset
